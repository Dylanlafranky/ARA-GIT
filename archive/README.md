# Archive

This directory contains the exploratory research trail of the ARA framework project. These files have been moved out of the repo root to reduce clutter now that the canonical implementations live in `TheFormula/`.

Nothing here has been deleted — git history is preserved via `git mv`.

## Contents

### `numbered_tests/`
All 365+ numbered Python scripts (01_ through 243+variants) that represent the chronological research journey, from early harmonic ratio tests through system mapping, topology exploration, and formula engine development. Also includes associated data files (JSON), figures (PNG), and specification documents (MD) that were numbered as part of the sequence.

See `SUPERSEDED.md` in the repo root for a mapping of which scripts have been replaced by `TheFormula/` equivalents.

### `session_notes/`
Daily session notes (SESSION_NOTES_*.md) documenting research progress and decisions.

### `early_visualizations/`
HTML-based interactive visualizations created during early exploration — ARA maps, 3D ring visualizations, analysis dashboards, and domain-specific plots. These predate the polished visualizations in `TheFormula/`.

### `early_papers/`
Draft papers and preprints from "The Geometry of Time" series (Papers 1–9), plus the ARA multi-system mapping preprint and proof roadmap documents. These represent the evolving written formulation of the framework.

### `misc_notes/`
Miscellaneous research notes, musings, and one-off documents that don't fit other categories: prediction summaries, epistemology notes, draft posts, and audit records.

### `old_git_mirror/`
An older copy of the repository structure that was nested inside the repo as a `GIT/` subfolder. Preserved here for reference.
