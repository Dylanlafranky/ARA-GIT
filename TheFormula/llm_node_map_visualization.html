LLM node map — Pythia-70M dynamical structure

2026-05-10 · 55 components (7 layer-norm + 48 attention heads), tracked over 250 autoregressive generation steps. Each component's activation became a time series. Pairwise correlation between every pair of components produced a 55×55 coupling matrix. The matrix was spectrally embedded into 2D so components that move together appear close together.

The framework's coupling-graph approach surfaces real interpretability findings on Pythia-70M without being told what to look for: dead layers, within-layer clusters, cross-layer circuits, and anti-phase pairs.

The map: each dot is one component

Each dot is one node (layer norm or attention head). Position is determined entirely by which other nodes it co-moves with — close dots have similar activation dynamics. Components in layers 4-6 collapse to the origin because their activations had zero variance during this generation: dynamically dead at this scale.

What the map shows

  • Layers 4-6 are dynamically dead. 24 of 55 components had zero variance in their activation during generation. They collapse to the origin in the embedding. This is itself an interpretability finding — these later layers aren't "doing nothing" but their max-attention pattern is so deterministic it doesn't fluctuate.
  • Within-layer head clusters. Layer 2's heads (H0, H1, H2, H5, H6) all correlate above 0.95 with each other. They form a tight cluster in the embedding. The framework calls this "co-oscillating components" — what mechanistic interpretability would call a parallel circuit.
  • Cross-layer circuits. Specific pairs of heads in different layers move in lockstep: L0 H6 ↔ L2 H3 (corr 0.986), L1 H0 ↔ L2 H2 (0.985), L0 H4 ↔ L1 H0 (0.983). These are candidate information-flow paths through the network.
  • Anti-phase relationships. Layer-norm 2 and 3 are strongly anti-correlated with several attention heads (-0.94 to -0.97). When residual stream norm rises, those heads' attention focus diffuses. The framework calls this a matched-rung pair structure — two halves of one closed system.

Components in the same layer move together more than components in different layers — which makes architectural sense (same layer shares input and parallel attention) and is exactly what mechanistic interpretability would predict but rarely measures this way.

What this means

This is a mechanistic interpretability tool the framework gives essentially for free. Run any LLM, capture per-component activations during generation, compute the coupling matrix, embed it. You get:

  • A map of the model where components that work together cluster together.
  • Identification of dead components (zero-variance during this run).
  • Identification of circuits (cross-layer high-correlation paths).
  • Identification of matched-rung pairs (anti-phase coupled components).
  • A measurement of how localised computation is (within-layer vs across-layer correlation ratio).

None of this required knowing what attention heads "are supposed to" do. The framework's own dynamical-coupling lens surfaced the structure.

Caveats: Single generation run, single model, single seed. The "dead layers 4-6" finding might be specific to this prompt or this generation pattern; needs replication across many seeds and prompts. Some of the strong correlations might be driven by overall norm trends rather than fine-grained dynamical coupling — partial-correlation analysis (controlling for global drift) would tighten the claim. The coupling-matrix approach is well-known in network neuroscience; the contribution here is applying the framework's specific definition of coupling (correlation, anti-phase, ARA-coherence) and getting interpretable results on a real LLM. Bigger models and prompt diversity needed before claiming generality.
