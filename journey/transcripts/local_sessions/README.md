# Local Cowork session transcripts

These are unfiltered transcripts of the Cowork sessions in which the ARA framework was developed. They're posted here for full auditability — anyone who wants to verify that the published findings were arrived at honestly, rather than written up after the fact, can read the raw record.

A few things to know before reading:

- **This is research thinking in real time.** It includes enthusiasm, dead ends, course corrections, half-formed ideas that didn't pan out, and the occasional wrong turn that needed walking back. It is not academic prose.
- **The polished outputs are in the repo root** — `INDEX.md`, `MASTER_PREDICTION_LEDGER.md`, `what_is_this.html`, and the `release_2026-05/` directory are where the actual claims live. The transcripts are the underlying record, not the headline.
- **My voice is informal.** I'm not a trained scientist and I don't always use the right terminology. I've kept the transcripts unedited so the reasoning is verifiable, not so the prose looks good.

## What's here

- `*.jsonl` — raw Cowork session files, one per session, full tool calls and responses
- `*_readable.md` — markdown render of each main session for browsing (tool calls compressed)
- `INDEX.md` — chronological map of which session is which
- `subagents/` — transcripts of subagents spawned during the main sessions

## How this folder is meant to be used

The intended audience is someone who is already taking the framework seriously — has read the structured outputs, looked at the test scripts, and now wants to verify that the work happened the way the polished docs describe. If you've found this folder by browsing, that's the right starting position.

If you're a peer reviewer, journalist, or curious researcher, I'd rather you read the structured documents first and come here only if something there raises a question that the raw record can answer.
